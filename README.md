# Drainage Manhole Asset Management Dashboard (Power BI)

An end-to-end Power BI dashboard built to help infrastructure and municipal teams **identify risk, inspection gaps, and prioritization signals** across large drainage manhole networks.

This project focuses on **operational decision-making**, not surface-level reporting.

- Where are inspection gaps the largest?

## ğŸ¯ What This Dashboard Delivers

### 1. Inspection Coverage vs Backlog
- Clear comparison of inspected vs uninspected assets
- Visibility into inspection debt across municipalities and districts

### 2. Asset Aging & Risk Exposure
- Age-band analysis with emphasis on **40+ year old infrastructure**
- Identification of aging clusters instead of isolated old assets

### 3. High-Risk Asset Concentration
- Municipality-level and age-band level risk concentration
- Helps avoid spreading inspection effort thin

### 4. Traffic-Sensitive Asset Density
- Manhole concentration by highways and traffic-heavy zones
- Highlights assets requiring traffic control during maintenance

### 5. Actionable KPIs
- Inspection Coverage %
- Inspection Backlog Size
- Average Asset Age
- Traffic Control Requirement %
